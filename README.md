# my-home

Проект двухэтажного дома **13 × 14 м** с залом-студией у фасада (80 м², двусветный),
Армения. Смета «серой коробки», объёмы материалов, цены (list.am и др.) и интерактивная
визуализация с калькулятором.

## 🌐 Визуализация (GitHub Pages)

Интерактивная страница со схемами и сметой: **[index.html](index.html)**

После включения Pages доступна по адресу:
`https://mailtoharutyunyan.github.io/my-home/`
