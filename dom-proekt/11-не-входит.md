# 11 · Что НЕ входит в смету коробки

← [Оглавление](00-ОГЛАВЛЕНИЕ.md)

**Серая коробка** (земляные + материалы + работа) — это ≈19,7 млн драм. Всё ниже —
доводит дом до **«под ключ»** и уже посчитано в калькуляторе (вкладка «Смета»,
разделы «Отделка/инженерия» и «Благоустройство», пресеты Эконом/Стандарт/Премиум).

| Статья | Ориентир, драм | Примечание |
|---|---|---|
| Гидроизоляция + утепление плоской кровли | ≈1 500 000 | 182 м² × ~8 000; обязательно |
| Окна и двери | по проекту | зависит от размеров и материала |
| Штукатурка / стяжка / шпаклёвка | отдельная смета | черновая отделка |
| Финишная отделка (пол, стены, потолок) | отдельная смета | плитка, ламинат, краска |
| Электрика (проводка, щиток, розетки) | отдельная смета | |
| Сантехника (трубы, разводка, приборы) | отдельная смета | |
| Отопление (котёл, тёплый пол/радиаторы) | отдельная смета | |
| Фасад (утепление + облицовка) | отдельная смета | |
| Лестница (отделка, ограждение) | отдельная смета | сама конструкция — в коробке |
| Проект + разрешение на строительство | по договору | **обязательно**, сейсмика |
| Тепловой насос / солнечные панели | опция в калькуляторе | +2,5 млн / +350 тыс за кВт |
| Забор, ворота, септик, отмостка, двор | в калькуляторе | раздел «Благоустройство» |
| Подключение коммуникаций (вода, газ, свет, канализация) | по тарифам | |
| Проект + разрешение на строительство | по договору | **обязательно**, сейсмика |

> Полный расчёт «под ключ» (≈46,9 млн драм в стандартной комплектации) — в
> калькуляторе на сайте. Меняйте пресет и опции — итог пересчитывается.
