## Пятно застройки на участке (20 × 25 м)

Фасад дома — по стороне **20 м** (передняя сторона). Дом стоит **в глубине
участка**, отступ **3 м** от боковых и задней границ. Перед домом — двор ~9 м.

```
        ФАСАД (передняя сторона, 20 м)
 ┌───────────────────────────────────────┐
 │                                        │
 │        ДВОР / ПАРКОВКА  (~9 м)         │  ← вход, подъезд
 │                                        │
 │   ┌───────────────────────────┐  ↑     │
 │3м │                           │3м│     │
 │←→ │     ДОМ  14 × 13 м        │←→│ 25 м │
 │   │     182 м² / этаж         │  │     │
 │   │                           │  ↓     │
 │   └───────────────────────────┘        │
 │              ↕ 3 м (задний отступ)     │
 └───────────────────────────────────────┘
              ← 20 м →   (14 + 3 + 3 = 20)
```

- Ширина по фасаду: **14 м** + по 3 м с боков = 20 м ✓
- Глубина: **13 м** + 3 м сзади = 16 м → **двор перед домом ≈ 9 м**
- Дом занимает ≈36 % участка; двор впереди ~180 м² под парковку/подъезд.

> Если фасад должен быть 13 м (а 14 м — глубина), боковые отступы станут 3,5 м —
> скажите, поменяю ориентацию.

## Разрез — двусветный зал

```
        ┌──────────────────────────────────────┐  ← плоская крыша (монолит 16 см)
        │  спальни / комнаты 2-го этажа         │
   6 м  │            ┌───────────────┐          │  ← перекрытие 2 эт. (монолит 16 см)
        │  2 эт.     │               │          │
        │────────────┤     ЗАЛ       │──────────│  ← НЕТ плиты над залом
        │  1 эт.     │   H = 6 м     │          │
        │            │               │          │
        └────────────┴───────────────┴──────────┘  ← монолитная плита 30 см
        ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  подбетонка 10 см + щебень
```

## Ориентировочная разбивка площадей

| Уровень | Площадь | Примечание |
|---|---|---|
| 1 этаж | 182 м² | зал-студия + кухня, мастер-спальня, гостевой с/у, вход, лестница |
| 2 этаж | ≈102 м² | минус проём зала (80 м²): спальни, с/у, галерея |
| **Итого полезная** | **≈284 м²** | по этажам |

## Планировка (принцип)

- **Зал у фасада**, 8 × 10 = 80 м², двусветный (H 6 м), панорамные окна на двор.
- **Кухня-студия** объединена с залом через **раздвижную перегородку** (можно разделить),
  высота кухни 3 м (под перекрытием 2-го этажа).
- На 1-м этаже — **мастер-спальня** с гардеробной и **гостевой с/у** у входа.
- На 2-м этаже — спальни и с/у; над залом проём (галерея), плиты нет.
